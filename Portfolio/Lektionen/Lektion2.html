Lektion 2 – Grundlagen der JavaScript-Syntax

Einführung

Willkommen zur zweiten Vorlesung im Modul Webdevelopment! In dieser Einheit behandeln wir die Grundlagen der JavaScript-Syntax – von Entwicklungsumgebungen über Variablen, Datentypen, Kontrollstrukturen, Arrays, Objekte, Funktionen bis hin zu höheren Funktionen.

Was macht der Webbrowser?

  • Der Server serviert Webseiten und sendet sie an den Browser.
  • Der Browser ruft die Seite ab und zeigt die HTML-Struktur an.

HTML – HyperText Markup Language

  • Strukturiert Webseiten und beschreibt deren Inhalt.
  • Übersetzt Tags und deren Inhalt in sichtbare Elemente.
  • Überschriften reichen von <h1> bis <h6>.
  • Kommentare: <!-- Dies ist ein Kommentar -->
  • Manche Elemente (z. B. <img>) besitzen kein End-Tag.

CSS – Cascading Style Sheets

CSS verleiht HTML-Seiten Stil und Layout.

p {
  background-color: yellow;
  color: blue;
  border: 1px solid black;
}

Wird im <head>-Bereich mit dem Element <style type="text/css"> oder über externe Stylesheets eingefügt.

Entwicklungsumgebung

  • Visual Studio Code: Open Source, plattformunabhängig und erweiterbar.
  • Alternativen: Atom, Brackets, Notepad++, Bbedit.
  • Node.js: Führt JavaScript serverseitig aus.

JavaScript – Variablen und Datentypen

  • Dynamisch typisiert – Variablen können ihren Typ ändern.
  • Kann verwendet werden, um Textinhalte oder Layouts einer Seite zu verändern.
<script src="verzeichnis/datei.js"></script>

Variablendeklarationen

  • let: Lokaler Gültigkeitsbereich (empfohlen)
  • var: Global oder lokal (veraltet)
  • const: Konstante Werte

Datentypen: Number, String, Boolean, Object, null, undefined

Werte, Typen und Operatoren

  • Arithmetische Operatoren: +, -, *, /
  • Vergleichsoperatoren: ==, ===, !=, !==
  • Logische Operatoren: &&, ||, !
let a = 5;
let b = 10;
console.log(a + b); // Ausgabe: 15

Kontrollstrukturen

  • if, else, else if: Bedingte Anweisungen
  • for, while, do-while: Schleifen
for (let i = 1; i <= 5; i++) {
  console.log(i); // Ausgabe von 1 bis 5
}

Kontrollausgaben in der Konsole

MethodeBeschreibung
console.clear()Bereinigt die Konsole.
console.debug()Debugging-Informationen.
console.error()Fehlermeldung.
console.info()Allgemeine Informationen.
console.log()Standardausgabe.
console.trace()Stack-Trace.
console.warn()Warnmeldung.

Arrays

const fruits = ['Apple', 'Banana', 'Cherry'];
console.log(fruits[0]); // 'Apple'

fruits.push('Grapes');
console.log(fruits); // ['Apple', 'Banana', 'Cherry', 'Grapes']

String-Verkettung

let greeting = "Good";
let timeOfDay = "Morning";
let result = greeting + " " + timeOfDay; // "Good Morning"

Objekte

const book = {
  title: '1984',
  author: 'George Orwell',
  pages: 328,
  read: function() {
    console.log('Reading 1984...');
  }
};
console.log(book.title);
book.title = 'Animal Farm';

Funktionen

Traditionelle Definition

function add(a, b) {
  return a + b;
}

Pfeilfunktionen (ES6)

const add = (a, b) => a + b;

Pure Function

const double = (x) => x * 2;

Höhere Funktionen

const numbers = [1, 2, 3];
const doubled = numbers.map((n) => n * 2);
console.log(doubled); // [2, 4, 6]
